백엔드 서버:
1. 터미널 또는 명령 프롬프트를 엽니다.
2. 백엔드 디렉토리로 이동합니다: `cd C:\Users\flux304\Desktop\asdasd\pybo\backend`
3. 의존성을 설치합니다 (아직 설치하지 않았다면): `npm install`
4. 백엔드 서버를 시작합니다: `npm start`
   다음과 같은 메시지가 표시되어야 합니다: `Server is running on http://localhost:3001`

프론트엔드 서버:
1. 새 터미널 또는 명령 프롬프트를 엽니다.
2. 프론트엔드 디렉토리로 이동합니다: `cd C:\Users\flux304\Desktop\asdasd\pybo\frontend`
3. 의존성을 설치합니다 (아직 설치하지 않았다면): `npm install`
4. 프론트엔드 개발 서버를 시작합니다: `npm start`
   일반적으로 `http://localhost:3000` (또는 사용 가능한 다른 포트)에서 애플리케이션이 실행되는 새 브라우저 탭이 열립니다.

중요:
- 프론트엔드 서버를 시작하기 전에 백엔드 서버가 실행 중인지 확인하십시오.
- 문제가 발생하면 터미널 출력에서 오류 메시지를 확인하십시오.
